# Claude Code — Configuración del Proyecto SOLUTUM

Lee `docs/agents.md` para decisiones de este proyecto.
Lee `docs/guia.md` para principios generales.

**Jerarquía:** agents.md > guia.md

## Stack

- Tipo A avanzado: HTML multipágina + Vite + CSS nativo moderno + Vanilla ES Modules
- Fuentes: @fontsource/inter + @fontsource/space-grotesk (self-hosted, sin Google Fonts CDN)
- Lightbox: glightbox — solo en trabajos.html/trabajos.js
- Validación: zod cliente (Fase 1 visual, sin envío real)
- Hosting: Cloudflare Pages

## Reglas

- Mobile-first siempre
- Accesibilidad obligatoria (alt descriptivos, contraste 4.5:1, foco visible)
- INP < 200ms
- Self-hosted fonts — NO Google Fonts CDN
- CSS moderno nativo (nesting, container queries, scroll-driven animations)
- UTF-8 estricto — nunca convertir á/é/í/ó/ú/ñ a entidades HTML

## Zonas Rojas — FRENAR antes de:

- Ejecutar deploy
- Conectar formulario a backend
- Instalar dependencias fuera de la lista aprobada: vite, @fontsource/inter, @fontsource/space-grotesk, glightbox, zod
- Agregar React, Vue, Astro, Next.js, Tailwind, jQuery
- Modificar scripts de encoding existentes en scripts/
- Borrar o mover archivos en datos/

## Comandos útiles

```bash
npm run dev              # Servidor local Vite
npm run build            # Check encoding + build a dist/
npm run preview          # Preview del dist/
npm run check:encoding   # Detectar mojibakes
npm run fix:encoding     # Preview correcciones
npm run fix:encoding:apply  # Aplicar correcciones
```
